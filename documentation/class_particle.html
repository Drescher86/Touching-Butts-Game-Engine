void Particle::animateParticleColor ( ParticleAnimation  type,
glm::vec4  color1 = glm::vec4(),
glm::vec4  color2 = glm::vec4(),
glm::vec4  color3 = glm::vec4(),
glm::vec4  color4 = glm::vec4() 
)

Animate particle color through its life time, using a specific interpolation type. FIXED_STATIC only color1 is used. RANDOM_STATIC uses none of the colors. LINEAR_DYNAMIC only color1 and color2 are used. SPLINE_DYNAMIC all colors are used.

Parameters
typeParticleAnimation type.
color1Animation start color. Defaults to black.
color2Animation second color. Defaults to black.
color3Animation third color. Defaults to black.
color4Animation fourth color. Defaults to black.
